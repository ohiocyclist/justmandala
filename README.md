# justmandala
Updated version of the Just Mandala project found here on GitHub

Like the other project, makes Mandalas using symmetry.

Also has a color chooser, a color importer from the image of your choosing, a random mandala maker, and an auto fill system

Uses React and Bootstrap

setup with node/npm,
npm i
npm run dev

Put a toe in the water on unit testing with jest, npm test
